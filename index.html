File Metadata Service

A microservice to return a description of the metadata of an uploaded file.

To use, upload a file in the form below. The response will be a JSON object containing some metadata from the file.

example
// upload image
ukintelligencepuzzle.png

// response
{
name: ukintelligencepuzzle.png,
filesize: 63485,
mime-type: image/png
}
Created by Andrew Marshall with Node.js and Express.js. See it on Github!
